The Demographics of Chess (in Australia)

Description

This visualisation shows the number of titled chess players in each country, with a particular focus on Australia. The data is sourced from the FIDE (International Chess Federation) database, which provides information on chess players worldwide. The choropleth map uses a color gradient to represent the number of titled players in each country, allowing for easy comparison across regions.

Scatter Plot Description

The scatter plot visualises the relationship between the number of titled players and the population of each country. Each point on the plot represents a country, with the x-axis representing the population and the y-axis representing the number of titled players. This allows us to see if there is a correlation between a country's population size and its chess player base.

Pie Chart Description

The pie charts show the distribution of titled players by title type (e.g., Grandmaster, International Master, etc.) for Australia, the United States, India, Russia, and Germany. Each slice of the pie represents a different title type, and the size of each slice corresponds to the number of players holding that title in the respective country. This allows us to compare the composition of titled players across these countries.

Grouped Lollipop Chart Description

The grouped lollipop chart displays the number of titled female chess players in various countries, grouped by title type. Each group of lollipops represents a country, and within each group, the lollipops are colored according to the title type (e.g., Grandmaster, International Master, etc.). The length of each lollipop corresponds to the number of players holding that title in the respective country. This allows us to compare the distribution of titled female chess players across different nations.
